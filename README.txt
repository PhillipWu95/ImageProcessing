手写识别作业文件说明：
===========================data================================
--.\train_image\：训练用的手写数字图片（仅包含数字"1","6","8"）。

--.\test_image\：测试用的手写数字图片。

--train_labels.mat：训练数据的标签。其中索引序号（index）是训练数据的序号，即第几幅图片，
                    对应的数值为该图片所表示的实际数字。

--test_labels.mat：测试数据的标签，主要用于检测精确度。格式与上文训练数据相同。

--traindata.mat：训练数据整合。这里我们已经将所有数据整合到一起，后续程序直接基于它即可。
                 其中，X存储了所有训练图片的灰度值，每一行存储对应序号的训练图片数据，图片
                 存储格式为按列存储（图片灰度值一列一列地按顺序存成一行）。
                 Y即为训练数据的标签，与"train_labels.mat"中的格式相同。

--testdata.mat：测试数据整合。格式与上同。

===========================Matlab program================================
--viewdata.m：随机选取测试数据的100张图片进行显示。

--displayData.m：显示最终图片的函数，在viewdata中用到。

--img2data.m：将数据库中的图片转换成"traindata.mat"及"testdata.mat"的数据格式，方便后续实验
              的进行，最终结果已生成并提供出来，该程序仅用于辅助同学们进一步了解数据的存储
              格式。
